# Imersão Agentes de IA com n8n - Hashtag Treinamentos

Repositório com os projetos desenvolvidos durante a Imersão Agentes de IA com n8n da Hashtag Treinamentos. Este conteúdo representa meu aprendizado e experiências práticas adquiridas ao longo do treinamento intensivo sobre automação de workflows com inteligência artificial.

## Sobre a Imersão

A Imersão Agentes de IA com n8n é um programa prático e intensivo promovido pela Hashtag Treinamentos que ensina como criar agentes de inteligência artificial especializados para resolver problemas reais de negócios. Durante o treinamento, tive a oportunidade de aprender a construir soluções completas que combinam IA generativa com automação de fluxos de trabalho.

## Estrutura do Projeto

Este repositório está organizado em módulos sequenciais, cada um representando uma aula específica da imersão:

### [Aula 1 - Agente de IA Comercial para Atendimento via E-mail](./Imersão%20N8N%20HashTag/Aula%201/readme.md)
Primeiro contato com agentes de IA no n8n, criando um assistente comercial especializado em atendimento via e-mail. O agente monitora a caixa de entrada do Gmail e responde automaticamente a perguntas frequentes sobre cursos e serviços da Hashtag Treinamentos.

### [Aula 2 - Processamento de Reembolsos com Análise de Sentimento](/Imersão%20N8N%20HashTag/Aula%202/readme.md)
Desenvolvimento de um sistema automatizado para processar solicitações de reembolso recebidas por formulários web. O fluxo inclui extração de dados, análise de sentimento, consulta a base de clientes e respostas personalizadas por e-mail.

### [Aula 3 - Sistema de Atendimento com Agentes Especializados](/Imersão%20N8N%20HashTag/Aula%203/readme.md)
Criação de um sistema completo de atendimento automatizado com um agente gerente e três subagentes especializados (Financeiro, Suporte e Customer Success). O sistema opera via Telegram e distribui inteligentemente as requisições para os especialistas adequados.

### [Aula 4 - Processamento Automatizado de Faturas](/Imersão%20N8N%20HashTag/Aula%204/readme.md)
Implementação de um sistema para processar automaticamente faturas recebidas por e-mail. O fluxo extrai dados estruturados dos PDFs usando IA, classifica como corporativo ou pessoal, organiza os arquivos no Google Drive e registra informações em planilhas online.

## Tecnologias Utilizadas

- **n8n**: Plataforma de automação de workflows
- **Google Gemini**: Modelo de IA para processamento de linguagem natural (utilizado nas aulas 1, 2, 3 e 4)
- **Groq (OpenAI)**: Modelo de IA para processamento de linguagem natural (utilizado na aula 3 - subagentes CS e Financeiro)
- **Groq (Meta Llama)**: Modelo de IA para processamento de linguagem natural (utilizado na aula 3 - subagente Suporte)
- **Gmail API**: Integração com e-mails
- **Google Sheets**: Armazenamento e manipulação de dados
- **Google Drive**: Armazenamento de arquivos
- **Google Calendar**: Agendamento de reuniões
- **Telegram Bot API**: Integração com chatbot
- **Tally**: Plataforma de formulários web
- **Google Drive API**: Upload e organização de arquivos
- **Google Sheets API**: Manipulação de dados em planilhas
